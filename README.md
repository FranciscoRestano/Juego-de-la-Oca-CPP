# Juego de la Oca (Goose Game)

### Descripción del Proyecto
Implementación del clásico juego de mesa "El Juego de la Oca" desarrollada como proyecto académico.

### Características
* Interfaz basada en consola o aplicación simple.
* Lógica del juego implementada con Programación Orientada a Objetos (POO).
* Manejo de reglas de casillas especiales (Puente, Pozo, Cárcel, etc.).

### Tecnologías Utilizadas
* C++
* Qt (si aplica por los archivos .ui y .pro)
